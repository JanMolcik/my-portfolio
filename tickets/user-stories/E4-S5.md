## Goal
- As reader, I want writing detail richtext rendered semantically, so that long-form articles preserve headings, lists, quotes, links, inline code, code blocks, bold, and italic.

## Scope
- Server-safe richtext article renderer and terminal typography; supports BW-003.
- Delivery aligned with G-C execution group.

## Non-goals
- Unrelated module rewrites.

## Acceptance
- `/writing/[slug]` uses a server-safe Storyblok richtext renderer or narrow whitelist renderer, never unsafe raw HTML, and preserves terminal-noir long-form readability with accessible heading prefixes and focus-visible links.
