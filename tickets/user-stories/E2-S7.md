## Goal
- As author, I want Markdown drafts imported into Storyblok richtext, so that local writing workflow feeds the CMS source of truth.

## Scope
- Markdown authoring/import workflow; supports BW-006.
- Delivery aligned with G-B execution group.

## Non-goals
- Unrelated module rewrites.

## Acceptance
- Import tooling defaults to dry-run, requires explicit publish credentials/flag for writes, validates frontmatter/provenance, rejects or sanitizes raw HTML, converts the approved CommonMark subset, and is never imported by runtime `src/*` code.
